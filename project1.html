Personal Portfolio Website


Overview & Goals

Designed and developed a centralized professional hub to showcase my software engineering competencies. The primary objective was to build a highly accessible, responsive, and performance-driven web application that serves as the cornerstone of my digital identity.

Beyond a simple display of work, this project served as a rigorous exercise in writing semantic HTML, organized CSS, and clean JavaScript to ensure a seamless experience for potential employers and collaborators.


Design Philosophy

The aesthetic is driven by a "content-first" minimalist approach, utilizing the Poppins typeface for modern readability. I prioritized a distraction-free user interface (UI) with a neutral color palette, allowing the project content to take center stage.

User Experience (UX) was a critical focus; the single-page application (SPA) feel is achieved through smooth scroll-snap navigation, while a custom-built dark mode ensures visual comfort in various lighting environments.


Technical Challenges

Responsive Architecture: Ensuring fluid layouts across devices (from 4k monitors to mobile screens) required a "mobile-first" strategy. I utilized complex CSS Media Queries to dynamically adjust grid layouts and typography scales.

DOM Manipulation: Implementing the custom hamburger menu and theme toggle required precise state management using vanilla JavaScript. The challenge lay in ensuring these interactive elements remained accessible (using ARIA labels) and performant without relying on heavy front-end frameworks.


Key Features

  • Adaptive Responsiveness: Fluid layout adjustments for mobile, tablet, and desktop interfaces.
  • Dark/Light Mode: A persistent theme toggle implemented with CSS variables and JS state handling.
  • Semantic Accessibility: Built with semantic HTML5 tags and ARIA roles to support screen readers and SEO.
  • Interactive Navigation: Custom JS-driven mobile navigation with smooth scrolling anchor links.


Engineering Outcomes

This project solidified my grasp of the core web stack (HTML5, CSS3, ES6+ JavaScript).
